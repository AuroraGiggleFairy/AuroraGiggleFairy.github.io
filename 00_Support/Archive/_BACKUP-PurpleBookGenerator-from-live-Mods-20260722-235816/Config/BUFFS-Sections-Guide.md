## Section Titles

These titles are shared exactly with comments in buffs.xml.

### Purple Book Runtime Recalculate Buff

Top-level hidden recalculation buff. It initializes and refreshes all Purple Book cvars.

### Crafting Progress - Startup Rebuild

Startup rebuild for all crafting cvars.

### Crafting Total CVars Reset

Sets each crafting total cvar (craftingXCheck) to 0 before reseeding.

### Crafting Unlock Threshold CVars Reset

Sets each crafting threshold cvar (craftingXCheckN) to 0 before reseeding.

### Crafting Total CVars Seeded From ProgressionLevel

Replays each crafting skill level from ProgressionLevel requirements to restore current totals.

### Book Progress - Startup Rebuild

Startup rebuild for all book-series cvars.

### Book Series Totals Reset

Resets AGFskill* series counters to 0.

### Book Completion Flags Reset

Resets AGFperk*Complete flags to 0.

### Book Not-Complete Flags Reset

Initializes AGFNOTperk*Complete flags to 1.

### Book Perk Flags Reset

Resets AGFperk* per-book flags to 0.

### Book Series and Perk Flags Seeded From ProgressionLevel

Seeds AGFskill counters and AGFperk flags from vanilla perk progression values.

### Book Progress - Runtime Delta Updates

Runtime update group for book progression deltas.

### Book Series and Perk Flags Runtime Delta

Adds each newly unlocked book only once using AGFperk* Equals 0 guards.

### Book Completion Flags Runtime Update

Sets AGFperk*Complete when AGFskill* reaches that series book count.

### Book Not-Complete Flags Runtime Set

Turns lock-state cvars on while series is incomplete.

### Book Not-Complete Flags Runtime Clear

Turns lock-state cvars off once series is complete.

### Crafting Progress - Runtime Delta Updates

Runtime update group for crafting progression deltas.

### Crafting Unlock Threshold Runtime Update

Quality/tiered sections use one-active threshold bands; non-quality sections remain cumulative per unlock threshold.

### Crafting Total CVars Runtime Update

Updates crafting totals from ProgressionLevel so current skill levels stay mirrored.
