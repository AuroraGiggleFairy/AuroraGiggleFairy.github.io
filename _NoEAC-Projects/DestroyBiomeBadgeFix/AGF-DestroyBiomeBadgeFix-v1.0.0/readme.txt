AGF-DestroyBiomeBadgeFix v1.0.0

This mod fixes the BiomeBadge removal action so that it only affects badge slots, not clothing slots.

- Place the 'AGF-DestroyBiomeBadgeFix-v1.0.0' folder in your 7 Days To Die Mods directory.
- The fix is implemented via Harmony patch in Harmony/DestroyBiomeBadgeFix.cs.
- ModInfo.xml provides mod metadata.
