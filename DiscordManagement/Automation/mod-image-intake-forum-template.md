# Mod Image Intake - Forum Workflow

Use this with your Discord Forum channel `mod-image-intake`.

## Channel/Permission Intent

- Access: Survivor role can view and post (this is your "all people" group).
- Goal: each mod gets one forum post/thread for image intake and review.

Suggested Discord permission setup for this forum:
- `@everyone`: no access (optional, if you want Survivor-gated access only)
- `Survivor`: View Channel, Send Messages, Create Posts/Threads, Attach Files
- reviewer/admin roles: Manage Messages, Manage Threads, Manage Channel as needed

## Recommended Tags

- Needed
- Claimed
- Submitted
- Needs Retake
- Approved
- Done

## Required Submission Model

Each mod has two submission lanes:

1. Main
- Purpose: submit your best single image that showcases this mod clearly.
- This is the primary review lane.

2. General
- Purpose: optional extra/fun photos, alternate angles, cinematic moments, or bonus gameplay.
- These are welcome and can be approved independently of Main.

## Per-Mod Thread Template

Title suggestion:
`Help! Send image for mod MOD_NAME`

Body template:

```md
See image below for mod details.
Need an image or a collection of images that fit a 1920x1080 or similar 16:9 ratio.

File naming suggestion
- modname-lane-shotname-helper-date
- Example: agf-vp-farmingplus-main-corefeature-raven-2026-05-27
```

## Kickoff Message Template

```md
Image intake is now live in this forum.

How to contribute:
1. Open a mod thread.
2. Submit an image or image set that fits a 1920x1080 (16:9) slot.
3. Keep submissions in the correct mod thread so we know what each image is for.

All Survivor members can participate in this forum.
```
