## 3) Full Description (Copy/Paste)
Target: Nexus -> General -> Full description

```text
This mod expands craftable storage for 7 Days to Die Version 3 with larger capacity, practical pickup behavior, and shape variety designed for everyday gameplay.

Mod Type
- Server-Side (EAC-Friendly): Server install works for all joining players; EAC on or off.

Key Features
- Adds extra-large craftable storage blocks in multiple shapes.
- Storage blocks have 3x block health.
- Storage can be broken and picked back up.
- Vanilla capacities: Wood 60, Iron 80, Steel 100.
- With AGF backpack sizes 72+: Wood 120, Iron 144, Steel 168.

Compatibility
- Works standalone with no dependencies.
- Last tested on 7 Days to Die Version 3.
- Safe to install on an existing game.
- Dangerous to remove from an existing game.

Need Help?
- Discord support: https://discord.gg/Vm5eyW6N4r

Quick Install
1. Close the game.
2. Place the mod folder in the 7 Days to Die Mods folder.
3. Keep 0_TFP_Harmony installed.
4. Start the game or server and verify the mod loads.

Quick Remove
1. Back up your save first.
2. Remove the mod folder from Mods.
3. Start the game or server and verify behavior.
4. Because removal is marked dangerous, use a fresh save when possible.

Quick Update
1. Back up your save first.
2. Replace the old mod folder with the new version.
3. Remove older duplicate versions if present.
4. Start the game or server and verify the new version loads.
```
