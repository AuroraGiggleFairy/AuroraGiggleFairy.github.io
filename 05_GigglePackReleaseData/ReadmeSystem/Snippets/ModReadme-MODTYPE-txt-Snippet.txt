MOD TYPE 1
  - Mod Type: Server-Side (EAC-Friendly)
    - Server install works for all joining players.
    - EAC can be on or off.
    - Also works in singleplayer.

MOD TYPE 2
  - Mod Type: Server-Side (EAC Off)
    - EAC off required.
    - Server install works for all joining players.
    - Also works in singleplayer.

MOD TYPE 3
  - Mod Type: Server/Client-Side (Required)
    - EAC off required.
    - Host and all joining players must install it.
    - Also works in singleplayer.

MOD TYPE 4
  - Mod Type: Client-Side (Only)
    - EAC off required.
    - Server install has no effect.
    - Install on each player PC.
    - Also works in singleplayer.
