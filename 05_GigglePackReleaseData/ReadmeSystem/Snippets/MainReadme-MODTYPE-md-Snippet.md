MOD TYPE 1
- **Server-Side (EAC-Friendly)**: Server install works for all joining players, EAC can be on or off, and it also works in singleplayer.

MOD TYPE 2
- **Server-Side (EAC Off)**: EAC off is required, server install works for all joining players, and it also works in singleplayer.

MOD TYPE 3
- **Server/Client-Side (Required)**: EAC off is required, the host and all joining players must install it, and it also works in singleplayer.

MOD TYPE 4
- **Client-Side (Only)**: EAC off is required, server install has no effect, each player installs it on their own PC, and it also works in singleplayer.
