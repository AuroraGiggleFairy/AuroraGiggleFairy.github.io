[color=#8DB580][b][size=4]A. Install Mods[/size][/b][/color]
[list=1]
[*]------------------------------------------------------------------------[/*]
[*]Close the game.[/*]
[*]In Steam, right-click 7 Days to Die -> Manage -> Browse local[/*]
[*]files, then open Mods.[/*]
[*]Extract the zip into the Mods folder. Make sure it ends up as[/*]
[*]Mods/<ModName>/ModInfo.xml.[/*]
[*]Restart the game.[/*]
[*]------------------------------------------------------------------------[/*]
[/list]

[color=#8DB580][b][size=4]C. Backups[/size][/b][/color]
[list]
[*]------------------------------------------------------------------------[/*]
[*]To Create:[/*]
[*]Open %appdata% -> Roaming -> 7DaysToDie -> Saves, then open your[/*]
[*]World Name folder (for example, Navezgane).[/*]
[*]Copy your Game Name folder (for example, MyGame) to a safe place.[/*]
[*]To Restore:[/*]
[*]Copy that saved Game Name folder back into the same World Name[/*]
[*]folder in Saves.[/*]
[*]Replace the current folder if asked.[/*]
[*]------------------------------------------------------------------------[/*]
[/list]

[color=#8DB580][b][size=4]D. Update Mods[/size][/b][/color]
[list=1]
[*]------------------------------------------------------------------------[/*]
[*]Close the game.[/*]
[*]Make a backup first (see section C).[/*]
[*]Install the new version in Mods.[/*]
[*]If asked, allow overwrite or replace.[/*]
[*]If both old and new folders are there, keep the newer one and[/*]
[*]delete the older one.[/*]
[*]Start the game and confirm your save loads.[/*]
[*]------------------------------------------------------------------------[/*]
[/list]

[color=#8DB580][b][size=4]E. Remove Mods[/size][/b][/color]
[list=1]
[*]------------------------------------------------------------------------[/*]
[*][b]Warning:[/b] Removing a mod from an active save can destroy your saved[/*]
[*]game. Back up first.[/*]
[*]Never delete 0_TFP_Harmony; it comes with the game.[/*]
[*]Close the game.[/*]
[*]In Mods, delete each mod folder you are removing, except[/*]
[*]0_TFP_Harmony.[/*]
[*]------------------------------------------------------------------------[/*]
[/list]

[color=#8DB580][b][size=4]F. The 0_TFP_Harmony Mod (Do Not Remove)[/size][/b][/color]
[list=1]
[*]------------------------------------------------------------------------[/*]
[*]Never delete 0_TFP_Harmony; it comes with the game.[/*]
[*]If it is missing, restore it by verifying game files in Steam:[/*]
[*]In Steam, right-click 7 Days to Die.[/*]
[*]Select Properties.[/*]
[*]Select Installed Files.[/*]
[*]Click Verify integrity of game files and wait for completion.[/*]
[*]------------------------------------------------------------------------[/*]
[/list]

[color=#8DB580][b][size=4]G. EAC[/size][/b][/color]
[list=1]
[*]------------------------------------------------------------------------[/*]
[*]EAC stands for Easy Anti-Cheat and helps protect multiplayer[/*]
[*]sessions from cheating.[/*]
[*]Some mods require EAC to be turned off so they can work.[/*]
[*]How to launch 7 Days to Die with EAC off:[/*]
[*]In Steam Library, select 7 Days to Die.[/*]
[*]Click Play.[/*]
[*]In the launch popup, select Launch game without EAC.[/*]
[*]Click Play.[/*]
[*]If the launch popup does not appear:[/*]
[*]In Steam Library, select 7 Days to Die.[/*]
[*]Click the gear icon on the right, then click Properties.[/*]
[*]Under Launch Options, open the Selected Launch Option dropdown.[/*]
[*]Choose Ask when starting game or Launch game without EAC.[/*]
[*]If you run multiplayer with EAC off, use these safety practices:[/*]
[*]Simplest method: keep your server password private and have people[/*]
[*]ask for it.[/*]
[*]If you want tighter security on who joins, use the whitelist[/*]
[*]system.[/*]
[*]Admin tools such as Server Tools have security options.[/*]
[*]Talk to other server hosts, for example AGF in Discord:[/*]
[*]https://discord.gg/Vm5eyW6N4r[/*]
[*]------------------------------------------------------------------------[/*]
[/list]
