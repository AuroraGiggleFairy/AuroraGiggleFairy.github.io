========================================================================
                       AGF-BACKPACKPLUS-060SLOTS                        
========================================================================

60-slot backpack, encumbrance accounted for, no UI shrinking.


NOTE: AGF Mod Guide and Changelog are further below.


------------------------------------------------------------------------
MOD SCOPE
------------------------------------------------------------------------

  - Mod Version: 4.1.0
  - 7d2d Version: 3
  - Website: https://auroragigglefairy.github.io/
  - Languages Supported: All 13 game-supported languages.
  - Mod Type: Server-Side (EAC-Friendly)
    - Server install works for all joining players.
    - EAC can be on or off.
    - Also works in singleplayer.
  - Safe to install on existing game: Yes (Safe)
  - Safe to remove from existing game: No (Dangerous)
  - Dependencies: None, works standalone.


------------------------------------------------------------------------
FEATURES
------------------------------------------------------------------------

  - Backpack size is now 60 slots instead of 45, with no UI shrinking.
  - Carry capacity starts at 30 with 30 encumbrance slots.
  - Max Pack Mule fully unlocks encumbrance.
  - Buffs, Twitch integration, and mobility are accounted for.


------------------------------------------------------------------------
------------------------------------------------------------------------
OTHER DETAILS
------------------------------------------------------------------------

- Inventory cell size remains at vanilla size (67x67, no shrinking).
    - Pack Mule encumbrance relief per rank: 6, 12, 18, 24, 30 (full
      encumbrance unlock at max rank).
    - Encumbrance movement penalty multiplier is set to 0.0155 (default
      0.026) to preserve vanilla-style mobility behavior.
    - Vanilla mobility baseline: with all encumbrance slots filled,
      mobility is about 53% (about 23% with heavy armor).
    - Buffs, Twitch integration, and dropped backpack size are all
      accounted for.
    - Pack Mule localization is updated with an appended AGF detail
      line, with best effort for all game-supported languages.
