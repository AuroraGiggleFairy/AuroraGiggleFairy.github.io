3.  INSTALLATION NOTES
	*First, if you run into any conflicts, you may contact AuroraGiggleFairy via discord: https://discord.gg/Vm5eyW6N4r
		-All questions welcome from newcombers to seasoned 7d2d people.

	-REQUIRES EAC to be turned off.
	-This must be installed on the client to work.
	-All 13 languages supported
	-EnhancedPatch-ClientSideNoEAC is SAFE to install on new or existing games.
	-EnhancedPatch-ClientSideNoEAC is SAFE to remove from an existing game.
