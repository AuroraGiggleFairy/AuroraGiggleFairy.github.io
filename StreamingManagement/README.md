# Streaming Management

Administrative source and tooling for AGF's streaming setup. This domain is a
separate animal from the mod lifecycle, but it stays in this same workspace.
It is intentionally unnumbered at the repo root (not under `00_Support`),
mirroring `DiscordManagement/`.

## Current Setup

- **Platform:** [Twitch](https://www.twitch.tv/auroragigglefairy)
- **Automation/Chatbot:** Streamer.bot
- **Broadcast software:** OBS
- **Assets:** Own music and images (self-produced/owned)

## Scope

Nothing is built here yet. This folder exists to hold whatever we develop
together going forward, for example:

- Streamer.bot actions, commands, and config exports
- OBS scenes, sources, and scripts
- Overlay assets and image/music organization
- Stream automation tying into the mod release pipeline (e.g. announcing new
  GigglePack releases on stream)

Structure will grow as we decide what to add.
